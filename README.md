# Square1 - base code for a Metalsmith site

This is an example of an initial Metalsite. The build workflow uses Gulp and Metalsmith. All asset compilation is done with Gulp and the site is assembled with Metalsmith.

This project is a work-in-progress. As I build sites I'll expand on Square1. 

## Installation
````
git clone https://github.com/wernerglinka/square1.git
run npm install
gulp
````
